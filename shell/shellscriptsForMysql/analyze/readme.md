1. 获取所有某个binlog

   ```shell
   mysqlbinlog -vv --base64-output=DECODE-ROWS mastera.000028 > /root/mybinlog.txt
   ```

   ​

2. 复制一份binlog来分析

   ```shell
   cp /root/mybinlog.txt /root/mybinlog.txt.bac
   ```

   ​

3. 获取时间、位置、sql和语句

   ```shell
   # 截取所有sql带时间点和pos
   awk  '$0~/^###/ || $0~/end_log_pos.*flags/ {print $0}' mybinlog.txt.bac | sed 's/^### //;s@\/\*.*\*\/@@' > new


   ```
